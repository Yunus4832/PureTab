# PureTab Privacy Policy

PureTab does not collect, transmit, sell, or share personal data.

PureTab stores settings locally in your browser, including theme, background image, logo settings, and local search history. A third-party search engine selected for an individual search is not saved as the default for future new tabs. These values remain on your device.

If you enable bookmark suggestions, PureTab reads your browser bookmarks locally to show matching suggestions in the new tab search box. Bookmark data is not uploaded or sent to any server.

If you enable browser history suggestions and grant the optional history permission, PureTab reads your browser history locally to show matching suggestions in the new tab search box. Browser history data is not uploaded or sent to any server.

PureTab only uses locally available browser data to provide the features you enable. It does not use this data for advertising, analytics, profiling, or any unrelated purpose.

Default searches use the browser Search API and the user's browser default search provider. If the user explicitly selects another provider for an individual search, the query is sent directly to that provider. PureTab does not proxy, log, or inspect those requests.
